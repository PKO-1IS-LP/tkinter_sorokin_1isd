## 🎯 Задача 1: Яркое приветствие

Создание красочного окна с приветственным текстом в двух разных стилях.

### 📸 Ожидаемый результат:
```
┌─ Заголовок приложения ─────────────────┐
│                                        │
│          HELLO WORLD!                  │
│                                        │
│    Это мой первый яркий текст          │
│                                        │
└────────────────────────────────────────┘
```

### 📝 Реализация:

```python
from tkinter import *

# Создание главного окна
root = Tk()
root.title("Заголовок приложения")
root.geometry("500x200")
root.iconbitmap(default="")

# Настройка внешнего вида
root.configure(bg='#FFFACD')  # Светло-желтый фон

# Главный заголовок
label1 = Label(
    root,
    text="HELLO WORLD!",
    font=("Impact", 48, "bold"),
    fg="#800080",  # Фиолетовый текст
    bg="#FFFACD"   # Фон как у окна
)
label1.pack(expand=True)  # Центрирование

# Подзаголовок
label2 = Label(
    root,
    text="Это мой первый яркий текст",
    font=("Comic Sans MS", 24, "bold"),
    fg="#C71585",  # Розовый текст
    bg="#FFFACD"
)
label2.pack()

root.mainloop()
```

### ✨ Особенности:
- 🎨 Гармоничное сочетание цветов
- 📏 Разные размеры шрифтов для создания иерархии
- 🖼️ Метод `expand=True` для идеального центрирования
- 💫 Использование разных стилей шрифтов

## 🎯 Задача 2: Три цветных блока

Создание окна с тремя цветными фреймами, расположенными горизонтально.

### 📸 Ожидаемый результат:
```
┌─ Три блока ────────────────────────────┐
│                                        │
│   ┌────────┐  ┌────────┐  ┌────────┐  │
│   │        │  │        │  │        │  │
│   │ Текст  │  │        │  │        │  │
│   │        │  │        │  │        │  │
│   └────────┘  └────────┘  └────────┘  │
│                                        │
└────────────────────────────────────────┘
```

### 📝 Реализация:

```python
from tkinter import *

root = Tk()
root.title("Три блока")
root.geometry("600x300")
root.configure(bg="#FFFFFF")  # Белый фон

# Контейнер для горизонтального расположения
container = Frame(root, bg="#FFFFFF")
container.pack(expand=True)

# Красный блок
frame1 = Frame(container, bg='#FF4040', width=180, height=260)
label = Label(
    frame1, 
    text="Текст", 
    bg='#FF4040', 
    fg='white', 
    font=('Arial', 12, 'bold')
)
label.place(relx=0.5, rely=0.5, anchor='center')
frame1.pack(side='left', padx=10)
frame1.pack_propagate(False)  # Фиксируем размер

# Синий блок
frame2 = Frame(container, bg='#4040FF', width=180, height=260)
frame2.pack(side='left', padx=10)

# Зеленый блок
frame3 = Frame(container, bg='#40FF40', width=180, height=260)
frame3.pack(side='left', padx=10)

root.mainloop()
```

### 🎯 Ключевые моменты:
- 🧩 Использование контейнера для группировки
- 📐 `pack_propagate(False)` для фиксации размеров
- 🎯 `place()` для точного центрирования текста
- 📏 `side='left'` для горизонтального расположения
- 🖼️ Отступы `padx=10` между блоками

## 🔧 Полезные методы

### 📦 Методы pack():
- `pack(expand=True)` - центрирование виджета
- `pack(side='left')` - размещение слева
- `pack(side='right', padx=10)` - размещение справа с отступом
- `pack(fill='both')` - растягивание по обеим осям

### 📍 Методы place():
- `place(relx=0.5, rely=0.5, anchor='center')` - точное центрирование
- `place(x=10, y=20)` - абсолютное позиционирование
- `place(relwidth=0.5, relheight=0.3)` - относительные размеры

### 🖼️ Работа с Frame:
- `pack_propagate(False)` - запрет на изменение размера
- `configure(bg='color')` - изменение цвета фона
- `width=180, height=260` - фиксированные размеры
